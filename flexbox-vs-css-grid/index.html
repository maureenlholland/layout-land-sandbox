Flexbox vs CSS Grid (but not really - you can use both!)

CSS Grid

CSS Grid layouts are always thinking in two-dimensions: rows and columns. It can specify how a particular item aligns with its particular row and column. Also, the order does not matter. The item goes where it is told to go.

Flexbox

Flexbox is always thinking in one direction (according to the flex-direction property): rows or columns. It can wrap into multiple rows or columns if needed, but is never positioning based on the particular row or column. Order can be reversed, or re-arranged with the flex-order property on the flex-item.
